# Workshop AdonisJS

Aplicação de exemplo utilizando AdonisJS ministrado por Flávio Filipe [@flaviofilipe](https://github.com/flaviofilipe)

## Instalação

Copiar ```.env.example``` para ```.env```
Copiar ```server/.env.example``` para ```.env```

## Rodando o projeto

```
docker-compose up
```
